<h1 align="center">浮之静</h1>

> 仓库名由来：取首字母 `FZJ`，因过于繁琐，故取其中间字母 `Z`，`Z` 是一个很有意思的字母，像弹簧，蓄力愈久，反弹的力量就愈大。很符合 «浮之静» 之本意。
